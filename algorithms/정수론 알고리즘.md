# 기본 개념

소수(prime number)
: 양의 약수로 1과 자기자신만을 가지는 자연수를 말한다. 

합성수(composite number)
: 소수의 반댓말로 세 개 이상의 양의 약수를 갖는 자연수를 합성수라고 한다. 예를 들어, 9는 1, 3, 9의 약수를 가지므로 합성수이다.
