# publishOn

간단히 말해 런타임 실행의 일부를 지정된 워커로 이동할 수 있습니다. 

리액터는 런타임에 데이터를 처리할 워커를 지정하기 위해 스케쥴러라는 개념을 도입했습니다. 

이 스케쥴러는 인터페이스 인데, 이 인터페이스가 현재 스트림에 대해서 특정 워커를 선택하는 용도로 사용됩니다. 

```
Scheduler scheduler = 

Flux.range(0, 100)
	.map(String.valueOf)
	.filter(s -> s.length() > 1)

	.publishOnm(scheduler) // 
	

```
