# 스프링 프레임워크


## 스프링 코어(DI, AOP)

스프링 코어 중에서 가장 중요한 것은 DI(dependency injection)과 AOP(aspect oriented programming)이다.

DI의 경우, 하나의 애플리케이션을 동작하기 위해 여러개의 컴포너느를 통합해서 사용한다. 이때 의존성 주입이라는 접근방식이 큰 힘을 발휘한다.
