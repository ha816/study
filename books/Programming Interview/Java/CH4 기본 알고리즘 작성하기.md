# 리스트 정렬하기

>Comparable과 Comparator 인터페이스의 차이는?

자바는 정렬 순서를 정하기 위해서 두 가지 인터페이스를 제공한다. 

Comparable은 비교하려는 객체간의 자연스러운 순서로 정렬할때 사용하고 Comparator는 원하는 대로 정렬 순서를 정하고 싶을 때 사용한다.
