# 리스트 정렬하기

>Comparable과 Comparator 인터페이스의 차이는?

자바는 정렬 순서를 정하기 위해서 두 가지 인터페이스를 제공한다. 

Comparable은 비교하려는 클래스가 직접 구현하는 인터페이스로 객체간에 비교에 주로 사용한다. 그에 반해서 Comparator는 대상 객체들을 입력 받아 비교하는 주체로 사용한다.
