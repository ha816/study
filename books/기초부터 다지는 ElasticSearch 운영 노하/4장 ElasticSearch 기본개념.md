# 클러스터와 노드의 개념

먼저 클러스터와 노드에 대해서 살펴보자. 클러스터란 여러 대의 컴퓨터 혹은 구성요소들을 논리적으로 결합하여 전체를 하나의 컴퓨터 혹은 하나의 구성요소처럼 사용할 수 있게 해주는 기술이다. 마찬가지로 ElasticSearch 클러스터 역시 여러 개의 ES 프로세스들을 논리적으로 결합하여 하나의 ES 프로세스처럼 사용할 수 있게 해준다.
