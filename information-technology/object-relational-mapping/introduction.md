# Overview

JPA는 Java Persistence API의 약자로 자바 객체와 DB 테이블 간의 매핑을 처리하기 위한 ORM(Object-Relational Mapping) 표준이다. 

JPA 스펙 구현체를 JPA 프로바이더라고 하며, 하이버네이트, 이클립스링크 등이 있다. 그 중 하이버네이트를 많이 들 사용하는데 그 이유는 인기가 많기 때문이다.
